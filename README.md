Open source unmanned surface vehicle

<ins>*Hardware*</ins>
- ELRS ER5A Receiver
- Brushless Motor A2212 KV2000
- ESC Simonk 20A
- Servo 9SG
- LiPo Battery 2200MAh
- Action Cam B-PRO 5

<ins>*Mechanical*</ins>
- 3D Printed PLA <br>
